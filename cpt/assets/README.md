# CPT art asset map

The game now uses file-based sprites/icons so you can directly swap in your pixel-art exports.

Replace these files (keeping file names) with the provided UI artwork:

- `house.svg`
- `car.svg`
- `van.svg`
- `player-front.svg`
- `agent.svg`
- `icon-calendar.svg`
- `icon-coins.svg`
- `icon-shield.svg`
- `icon-search.svg`

> Tip: If your source files are PNG/WebP, either convert them to SVG-compatible names or update paths in `cpt/style.css`.
